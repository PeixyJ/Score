# 实时评分系统

一个支持多人评分的实时系统，包含手机评分端和PC大屏展示。

## 功能特性

- 多评分者支持，每人通过唯一密钥识别
- 多维度评分，维度可自定义配置
- 多人对同一人打分时自动计算平均分
- PC大屏实时显示排名，分数变化时自动排序
- 数据持久化存储（SQLite）

## 快速开始

### 1. 启动后端服务

```bash
cd server
npm install   # 首次运行需要安装依赖
npm start
```

后端将在 `http://localhost:3000` 启动，控制台会显示局域网访问地址。

### 2. 启动前端服务

```bash
cd client
npm install   # 首次运行需要安装依赖
npm run dev
```

前端将在 `http://localhost:5173` 启动。

## 访问地址

| 页面 | 地址 | 说明 |
|------|------|------|
| 首页 | http://[IP]:5173/ | 入口页面 |
| 管理后台 | http://[IP]:5173/admin | 配置评分者、被评分者、评分维度 |
| 手机评分 | http://[IP]:5173/mobile | 评分者使用手机扫码访问 |
| PC大屏 | http://[IP]:5173/screen | 实时显示排名 |

> 将 `[IP]` 替换为你电脑的局域网 IP 地址

## 使用流程

### 1. 初始配置（管理后台）

1. 访问管理后台 `/admin`
2. **添加评分者**: 输入名称后系统自动生成密钥
3. **添加被评分者**: 输入需要被评分的人员名称
4. **配置评分维度**: 设置评分项目（如：表现、创意、技术等），可调整满分和权重

### 2. 评分者打分（手机端）

1. 手机访问 `/mobile`
2. 输入分配给自己的密钥登录
3. 选择要评分的人员
4. 为各维度打分后提交

### 3. 大屏展示（PC端）

1. PC打开 `/screen` 全屏显示
2. 实时显示所有人的分数和排名
3. 当有新评分时自动更新并重新排序
